# BotMakers Assignment

A responsive, high-performance landing page built for the BotMakers internship selection.

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4
- **Animations**: GSAP (GreenSock Animation Platform) + ScrollTrigger + Lenis
- **Icons**: React Icons (Fa, Bs, Md, Lia, Gi)

## 🎨 Fonts Used

- **Orbitron**: For headings and esports-themed text.
- **Roboto**: For descriptions, inputs, buttons, and other body content.

## 🛠️ Installation & Setup

1. **Clone the repository** (or navigate to the project directory).
2. **Install dependencies**:
   ```bash
   pnpm install
   # or
   npm install
   ```
3. **Run the development server**:
   ```bash
   pnpm run dev
   # or
   npm run dev
   ```
4. **Build the production bundle**:
   ```bash
   pnpm run build
   # or
   npm run build
   ```
